# AWS-Resume-Website-Deployment-Using-EC2-Nginx-Cloudfront
📌 Project Overview:
Built and deployed my resume on an AWS EC2 instance using Nginx as a web server. The project demonstrates manual cloud infrastructure provisioning, Linux server configuration, and web server deployment to host a production-ready website.
This setup simulates real-world infrastructure deployment where a compute instance serves static web content directly without managed hosting services.
